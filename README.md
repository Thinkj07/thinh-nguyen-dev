# Thinh Nguyen Duc - Portfolio Website

A modern, responsive portfolio website showcasing Thinh Nguyen Duc's skills, projects, and professional experience as an AI Engineer and Full Stack Developer.

## 📋 Overview

This is a personal portfolio website for Thinh Nguyen Duc, a third-year Computer Science student at Ho Chi Minh City University of Technology. The website features a clean, modern design with sections for services, resume, skills, projects, blog posts, and contact information.

## ✨ Features

- **Responsive Design** - Fully responsive layout that works on desktop, tablet, and mobile devices
- **Interactive Navigation** - Smooth scrolling navigation with active link highlighting
- **Project Filtering** - Filter portfolio items by category (All, Web Design, Mobile Apps, Landing Pages)
- **Skills Showcase** - Visual progress bars for frontend and backend skills
- **Testimonials Slider** - Interactive carousel for client/project testimonials
- **Blog Section** - Display of latest blog posts
- **Contact Form** - Integrated with EmailJS for sending messages directly
- **Download CV** - Direct download link for resume/CV

## 🛠️ Technologies Used

### Frontend
- HTML5
- CSS3
- JavaScript (ES6+)

### Libraries & Frameworks
- **RemixIcon** - Icon library
- **Swiper.js** - Touch-enabled slider/carousel
- **MixItUp** - Portfolio filtering and sorting
- **EmailJS** - Email functionality for contact form
